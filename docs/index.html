Up and running in a minute

Install the extension

Load it as an unpacked extension while it's pre–Web-Store. It only runs on app.hyperliquid.xyz.

1

Download & build

Clone the repo and run npm install then npm run build to produce the dist/ folder.

2

Load unpacked

Open chrome://extensions, turn on Developer mode, click Load unpacked, and pick dist/.

3

Open Hyperliquid

Go to app.hyperliquid.xyz, click the Hypurr icon to open the side panel, and pick a coin. Done.
